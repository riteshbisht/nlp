This zip archive contains source code and release binaries for each of the
Double Metaphone implemnetations included in my "Implement Phonetic Matching ("Sounds-like") 
Name Searches" with the Double Metaphone Algorithm" article series.

Note that in order to experiment with these implementations, you will need the
dmetaphone_demo.zip archive, containing the source code and release binaries
for each of the sample applications included with the article series.

The contents of this archive are as follows:
* MetaphoneLib - C++ source code of the Double Metaphone template classes
* MetaphoneCOM - COM Automation wrapper around MetaphoneLib, exposing Double Metaphone
implementation for use by Visual Basic, and Windows scripting langauges
* Metaphone.NET - C# implementation of Double Metaphone for .NET
* XPMetaphone - SQL Server Extended Stored Procedure wrapper around MetaphoneLib,
exposing Double Metaphone implementation to SQL code
